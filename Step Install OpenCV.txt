#---------------------- INTALACAO OPENCV ----------------------#

1.) Baixe e instale Python 3 para Windows executando Windows x86-64 executable installer:
 	diretorio>python

	Python 3.7.3 etc

	>>> exit()

2) Instale OpenCV para Python dando os comandos:
	diretorio> pip install opencv-python
	diretorio> pip install opencv-contrib-python
	
Teste a instalação OpenCV:
	diretorio> python
	>>> import cv2
	>>> print(cv2.__version__)
	Deve responder algo como:
	4.1.0.25
	Para sair:
	>>> exit()

Teste a instalacao do aruco:
	diretorio> python
	>>> import cv2.aruco
	>>> 
	>>> exit()
